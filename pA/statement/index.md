今天優奈又在WORLD．FANTASY．ONLINE遊戲裡到地下城尋寶。  
她來到了一扇用特殊魔法鎖住的門前。  
門上面有 $N$ 個方塊，每個方塊上面都有一個整數 $a_i$，$1$ 到 $N$ 各出現一次。  
要打開這扇門的方法是，讓方塊上整數的由小到大排列。  
但是這些方塊只能使用魔法來移動，而且使用一次魔法只能拿起任意一塊方塊，然後放到最左邊。  
使用魔法需要耗費魔力值，因此優奈想要用最少的移動次數來達到條件，請問最少要移動幾次呢？  

\clearpage

## 輸入
第一行有一個整數 $N$，  
下一行有 $N$ 個整數 $a_i$。  

## 輸出
輸出最少的移動次數。  

## 輸入限制
 - $1 \leq N \leq 10^6$
 - $1 \leq a_i \leq N$
 - $a_i$ 皆相異

## 子任務
\subtasks

\clearpage

## 範例輸入1
```
5
1 2 3 4 5
```

## 範例輸出1
```
0
```

## 範例輸入2
```
5
1 4 2 3 5
```

## 範例輸出2
```
3
```
