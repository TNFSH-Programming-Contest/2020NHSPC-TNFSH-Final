# pC
有一張 $N$ 個點 $M$ 條邊的連通帶權無向圖 $G$。
試求以 $S$ 為起點到各點的最短路徑距離 ($S$ 到本身距離為 $0$)。

定義路徑距離 = 路徑上總邊權 $\times$ 路徑邊數。

\clearpage

## 輸入
第一行 $N,~M,~S$
接下來 $M$ 行為 $(u_i,~v_i,~w_i)$ 代表 $u_i$ 及 $v_i$ 之中有一條權重為 $w_i$ 的邊

## 輸出
輸出 $N$ 個數字，第 i 個代表 S 到 i 的最短路徑距離。

## 輸入限制
 - $1 \leq N \leq 2000$
 - $1 \leq M \leq 30000$
 - $1 \leq S,~u_i,~v_i \leq N$
 - $1 \leq w_i \leq 10^6$
